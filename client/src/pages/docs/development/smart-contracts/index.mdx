# Smart Contracts Development for AgriTrace

## Introduction

This repository contains the smart contracts for the AgriTrace project. The AgriTrace project is a blockchain-based solution for the traceability of agricultural products. The project is developed by the [AG Labs](https://agritrace.vercel.app) for final year project of the [Computer Science](https://jkuat.ac.ke) department at [SCIT](https://www.iitb.ac.in/).
